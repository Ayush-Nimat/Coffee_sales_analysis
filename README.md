# ☕ Coffee Sales Analysis Dashboard

An interactive Excel dashboard project focused on analyzing coffee shop sales data to uncover valuable business insights, sales trends, customer behavior, and product performance.

This project demonstrates practical data analytics and dashboarding skills using Microsoft Excel.

## 📌 Project Overview

The Coffee Sales Analysis Dashboard was created to transform raw sales data into meaningful visual insights that help understand business performance and support decision-making.

The dashboard provides:

- Sales performance tracking
- Product-wise revenue analysis
- Customer purchasing trends
- Peak sales period identification
- Interactive filtering and visualization 

## 🎯 Project Objectives
- Analyze overall coffee shop sales performance
- Identify top-selling products
- Discover peak business hours and days
- Compare category-wise revenue
- Create an interactive and visually appealing dashboard
- Generate actionable business insights
